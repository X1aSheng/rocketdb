## 三、Flash 布局分析

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           Flash 布局分析                                     │
├─────────────────────────────────────────────────────────────────────────────┤

【KVDB 扇区布局】

  扇区大小: 4096 bytes
  扇区头: 16 bytes (对齐后 16 bytes)
  数据区: 4096 - 16 = 4080 bytes

  ┌────────────────────────────────────────────────────────────────────────┐
  │ Sector Header (16 bytes)                                               │
  ├────────────────────────────────────────────────────────────────────────┤
  │ Offset │ Field      │ Size │ Value                                    │
  ├────────┼────────────┼──────┼──────────────────────────────────────────┤
  │ 0      │ magic      │ 4    │ 0x52444253 ("RDBS")                      │
  │ 4      │ erase_cnt  │ 4    │ 擦除次数                                 │
  │ 8      │ seq_id     │ 4    │ 扇区序列号                               │
  │ 12     │ reserved   │ 4    │ 0xFFFFFFFF                               │
  └────────┴────────────┴──────┴──────────────────────────────────────────┘

  ┌────────────────────────────────────────────────────────────────────────┐
  │ Record Header (16 bytes)                                               │
  ├────────────────────────────────────────────────────────────────────────┤
  │ Offset │ Field      │ Size │ Value                                    │
  ├────────┼────────────┼──────┼──────────────────────────────────────────┤
  │ 0      │ magic      │ 4    │ 0x52444252 ("RDBR")                      │
  │ 4      │ state      │ 2    │ 状态码                                   │
  │ 6      │ write_seq  │ 2    │ 写入序列号                               │
  │ 8      │ key_len    │ 2    │ key长度                                  │
  │ 10     │ val_len    │ 2    │ value长度                                │
  │ 12     │ key_crc    │ 2    │ key CRC16                                │
  │ 14     │ data_crc   │ 2    │ key+value CRC16                          │
  └────────┴────────────┴──────┴──────────────────────────────────────────┘

  状态转换 (NOR Flash 兼容):
    FREE    = 0xFFFF  →  (全1)
    ALLOC   = 0xFF00  →  (写入中)
    VALID   = 0xF000  →  (已提交)
    DELETED = 0x0000  →  (已删除)

  最小记录大小: 16(hdr) + 4(key_align) + 4(val_align) = 24 bytes
  最大记录数/扇区: 4080 / 24 = 170 条

【TSDB 扇区布局】

  扇区头: 28 bytes (对齐后 28 bytes)
  数据区: 4096 - 28 = 4068 bytes

  ┌────────────────────────────────────────────────────────────────────────┐
  │ TSDB Sector Header (28 bytes)                                          │
  ├────────────────────────────────────────────────────────────────────────┤
  │ Offset │ Field       │ Size │ Value                                   │
  ├────────┼─────────────┼──────┼─────────────────────────────────────────┤
  │ 0      │ magic       │ 4    │ 0x52445453 ("RDTS")                     │
  │ 4      │ erase_cnt   │ 4    │ 擦除次数                                │
  │ 8      │ time_start  │ 4    │ 首条记录时间                            │
  │ 12     │ time_end    │ 4    │ 末条记录时间                            │
  │ 16     │ seq_id      │ 2    │ 扇区序列号                              │
  │ 18     │ status      │ 2    │ WRITING/SEALED                          │
  │ 20     │ record_cnt  │ 2    │ 记录数量                                │
  │ 22     │ hdr_crc     │ 2    │ 头部CRC                                 │
  │ 24     │ reserved    │ 4    │ 保留                                    │
  └────────┴─────────────┴──────┴─────────────────────────────────────────┘

  ┌────────────────────────────────────────────────────────────────────────┐
  │ TSDB Record Header (8 bytes)                                           │
  ├────────────────────────────────────────────────────────────────────────┤
  │ Offset │ Field      │ Size │ Value                                    │
  ├────────┼────────────┼──────┼──────────────────────────────────────────┤
  │ 0      │ timestamp  │ 4    │ 时间戳                                   │
  │ 4      │ length     │ 2    │ 数据长度                                 │
  │ 6      │ crc        │ 2    │ 数据CRC (0xFFFF=未提交)                  │
  └────────┴────────────┴──────┴──────────────────────────────────────────┘

  最小记录大小: 8(hdr) + 4(data_align) = 12 bytes
  最大记录数/扇区: 4068 / 12 = 339 条
```
